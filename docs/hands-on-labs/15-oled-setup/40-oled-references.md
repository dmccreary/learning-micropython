!!! mascot-welcome "Explore and Learn More"

    You have finished the main OLED labs. Here are links to resources where you can go deeper. Explore at your own pace — there is always more to discover!

The links below will help you learn more about Organic Light-Emitting Diode (OLED) displays, display drivers, and advanced animation techniques.

1. [Analysis of OLED drawing performance using I2C interface](http://merkles.com/MediaWiki/index.php?title=SSD1306_128x64_I2C_OLED_Display) — this page measures how fast you can draw on an OLED using the Inter-Integrated Circuit (I2C) connection.
2. [Analysis of OLED drawing performance using SPI interface](http://merkles.com/MediaWiki/index.php?title=SSD1306_128x64_SPI_OLED_Display) — this page measures how fast you can draw on an OLED using the Serial Peripheral Interface (SPI) connection. SPI is much faster than I2C.
3. [Example of displaying a complex animation on an OLED with SPI](https://github.com/KonradStuewe/BadApple-ESP8266) — a project that plays a full video animation on a tiny OLED display. This shows how fast SPI can be!
4. [Video of the Above on YouTube](https://www.youtube.com/watch?v=7ZWeEIJo95U) — watch the animation running on a real device. It is amazing what a small screen can do.
5. [Peter Hinch's MicroPython Nano GUI](https://github.com/peterhinch/micropython-nano-gui) — a library that adds buttons, labels, and graphs to your OLED display. Great for building your own instruments and gadgets.
