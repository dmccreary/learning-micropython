# References: Wireless Connectivity and Internet of Things

1. [Internet of things](https://en.wikipedia.org/wiki/Internet_of_things) - Wikipedia - Explains how everyday devices connect and share data over networks. Sets the big-picture context for this chapter's wireless projects.

2. [Wi-Fi](https://en.wikipedia.org/wiki/Wi-Fi) - Wikipedia - Describes how devices join wireless networks, including SSIDs and access points. Directly supports the chapter's Pico W / ESP32 connection steps.

3. [MQTT](https://en.wikipedia.org/wiki/MQTT) - Wikipedia - Covers the lightweight publish/subscribe protocol popular for IoT messaging. Background for the chapter's data-sending examples.

4. MicroPython for the Internet of Things - Charles Bell - Apress - A project-driven book on connecting microcontrollers to networks and the cloud, closely matching this chapter's goals.

5. Building the Web of Things - Dominique Guinard & Vlad Trifa - Manning - Explains designing connected devices and simple web APIs, complementing the chapter's web-server work.

6. [network](https://docs.micropython.org/en/latest/library/network.html) - MicroPython - Official reference for the networking module used to manage Wi-Fi connections. The authoritative source for the chapter's connection code.

7. [network.WLAN](https://docs.micropython.org/en/latest/library/network.WLAN.html) - MicroPython - Reference for joining a Wi-Fi network and checking connection status. Directly supports the chapter's `connect()` examples.

8. [Raspberry Pi Pico W Web Server (MicroPython)](https://randomnerdtutorials.com/raspberry-pi-pico-web-server-micropython/) - Random Nerd Tutorials - Step-by-step guide to serving a web page that controls hardware from the Pico W. Mirrors this chapter's web-server project.

9. [MQTT.org](https://mqtt.org/) - MQTT - The official home of the MQTT standard with documentation and broker links. A reference for the chapter's IoT messaging discussion.

10. [Raspberry Pi Pico Series Documentation](https://www.raspberrypi.com/documentation/microcontrollers/pico-series.html) - Raspberry Pi - Official documentation covering the Pico W's wireless hardware. Authoritative background for the board used in this chapter.
